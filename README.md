# DizzyDashboard

A Progressive Web App for vestibular screening and BPPV treatment with particle physics simulation.

## Features
- Comprehensive 9-step clinical evaluation workflow
- BPPV treatment with canalith repositioning maneuvers
- Form data persistence across tab switches
- Chart ID generation and note export
- Responsive design optimized for mobile devices

## Technology Stack
- React with TypeScript
- Next.js
- Tailwind CSS
- Framer Motion
- Three.js for particle physics simulation

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
